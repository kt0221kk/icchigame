# 一致ゲーム

みんなで回答を一致させるオンラインゲームです。

## 特徴

- リアルタイムでマルチプレイヤーゲームを楽しめる
- お題はプレイヤーの投票で決定
- 回答が一致するとポイント獲得
- シンプルで直感的なUI

## ゲームの流れ

1. **ルーム作成/参加**: ホストがルームを作成し、他のプレイヤーはコードで参加
2. **お題提案**: 各プレイヤーがお題を提案
3. **投票**: 提案されたお題から投票で選択
4. **回答**: 選ばれたお題に対して回答を入力
5. **結果発表**: 一致した回答が表示され、ポイントが加算

## 技術スタック

- **フレームワーク**: Next.js 15 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **リアルタイム通信**: ポーリング（1秒間隔）
- **データ管理**: メモリ上（インメモリ）
- **デプロイ**: Vercel

## セットアップ

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開く

### ビルド

```bash
npm run build
npm start
```

## Vercelへのデプロイ

### 方法1: Vercel CLIを使用

```bash
npm install -g vercel
vercel
```

### 方法2: GitHubと連携

1. このリポジトリをGitHubにプッシュ
2. [Vercel](https://vercel.com)でGitHubリポジトリをインポート
3. 自動でビルド・デプロイされます

## 注意事項

- データはメモリ上に保存されるため、サーバー再起動で消失します
- 1時間以上更新がないルームは自動的に削除されます
- Vercelの無料プランで動作します
